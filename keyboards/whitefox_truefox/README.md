## Steps

1. Go to https://config.qmk.fm/
2. Load the correct keymap
3. Click `Compile`
4. Download the compiled firmware
5. Open `QMK Toolbox.app`
6. Load the firmware
7. Reset the keyboard
   1. WhiteFox has a physical button on the back


## Configurations

- **WhiteFox**: `input_club/whitefox` → `LAYOUT_65_ansi_blocker_split_bs`
- **Keychron Q1**: `keychron/q1v2/ansi_encoder`
  - Use `encoder` variant for knob support
  - Use `ansi` for US keyboard layout
